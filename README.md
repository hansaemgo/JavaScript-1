### 02. 변수(Variable) & 상수(Constant)

1. 변수선언(바뀔수 있는 값) : let value = 1;
2. 상수선언(값이 고정적) : const a = 1;
3. var 권장하지 않는다(같은 이름으로 여러번 선언 가능)
4. 데이터타입
    - 숫자(Number)
    - 문자열(String)
    - 참/거짓(Boolean) -> ture/false
    - null(진짜없다) undefined(아직정해지지않았다)
